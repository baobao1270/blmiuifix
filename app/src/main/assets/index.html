BiliBili HK 2.4.x navbar fix on MIUI 12

修复 BiliBili Legacy (2.4.x) 在未使用全面屏的 MIUI 12 上，三大金刚遮挡全屏播放器 UI 的问题。

使用说明

使用前请下载 Xposed。本人推荐使用 LSPosed。 使用前请务必阅读 LSPosed 的使用说明！ 安装好 LSPosed 和本应用后，在 LSPosed 模块中启用本应用并勾选港版 B 站即可。

机型适配：目前仅测试过 Redmi K30 Pro
注意： 仅支持 MIUI 12，且仅在 MIUI 12.5 开发版上测试过。其他版本可能有问题。
关于 B 站版本： 仅支持港版 B 站，版本为 2.4.0 或 2.4.2。
点我下载旧版 B 站

已知问题

在退出全屏后，需要对 UI 有任意一个操作，三大金刚才会显示出来。
